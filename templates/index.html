    新建爬虫任务

    支持正文页、目录页，或可直接下载的 txt 链接
    如果配置了网站规则，系统会按链接域名自动套用；也可以在这里手动指定。
    适用于像 cool18 这种版块首页列表页，会先提取帖子链接，再逐条创建抓取任务。
    仅在开启列表页批量抓取时生效，最大 50。

    新增站点规则

    支持单域名和通配子域名，例如 www.example.com、example.com、*.example.com
    支持多行填写，系统会按顺序尝试选择器。
    用于版块首页、列表页批量提取最近帖子；留空时不支持列表页批量抓取。
    用于正文页里继续抓同书其他分卷/章节帖子，并自动合并到同一本文本中。
    用于目录页提取章节链接；留空时使用通用链接识别逻辑。
    会在正文区域内移除这些元素，适合广告、工具栏、推荐区等。
    值越小优先级越高；多个规则命中同一站点时会优先使用。

    添加 AI 配置

    常用模型：gpt-3.5-turbo, gpt-4, gpt-4-turbo 支持手动填写模型，也支持测试连接后自动拉取模型列表
    OpenAI 和 Claude 需要提供 API Key，编辑时留空可保持原值，Ollama 可留空
    可选，用于自定义 API 地址或第三方兼容服务
    勾选后仅当前 AI 配置请求使用该代理；未勾选时会保留地址但不启用
    0-2 之间，值越高回答越随机
